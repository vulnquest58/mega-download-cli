# mega-download-cli

> Batch MEGA downloader powered by MEGA-CMD — with HackMyVM direct integration and automatic MEGA-CMD installer.

## Features

| Feature | Description |
|---------|-------------|
| 📥 **Batch download** | Download all MEGA links from a `links.txt` file |
| 📊 **Live progress bar** | Real-time speed, ETA, and file size |
| 🎯 **Smart selection** | `--select "1,3-5"` or `--level easy/medium/hard` |
| 🎮 **HackMyVM mode** | `--hmv <Machine>` resolves and downloads a HMV machine directly |
| 🔧 **Auto-installer** | Installs MEGA-CMD automatically if not found |
| 🔁 **Parallel workers** | `--jobs N` for concurrent downloads |
| 🗂️ **Failed log** | Failed URLs saved to `.mega_dl_logs/failed_links.txt` |

## Requirements

MEGA-CMD (`mega-get`) must be installed. The script will detect if it's missing and:
- **Linux**: run `apt install megacmd` or `snap install megacmd` automatically
- **Windows/macOS**: open the download page and print step-by-step instructions

For `--hmv` mode (HackMyVM integration), install:

```bash
pip install requests beautifulsoup4 rich
```

> Note: `vuln_hmv` must be available alongside this tool or in `$PATH` for `--hmv` mode to work.

## Installation

```bash
git clone https://github.com/vulnquest58/mega-download-cli.git
cd mega-download-cli
```

No additional setup needed for plain MEGA download (stdlib only).

## Usage

### Download from a links file

```bash
python mega_download.py -f links.txt -d ~/Downloads/VMs
```

### Interactive selection

```bash
python mega_download.py -f links.txt
# Shows numbered table → type: 1,3-5  or  all
```

### Filter by difficulty

```bash
python mega_download.py -f links.txt --level easy -d ~/VMs
```

### HackMyVM direct download

```bash
export HMV_USER=yourusername
export HMV_PASS=yourpassword

python mega_download.py --hmv Calculator -d ~/VMs
```

### List available links without downloading

```bash
python mega_download.py -f links.txt --list
```

## links.txt Format

Auto-generated by [`vuln_hmv`](https://github.com/vulnquest58/vuln_hmv) via `--save-links`. Format:

```
# MachineName  [Easy]  1.2 GB
https://mega.nz/file/XXXXXXXX#YYYYYYYYY

# AnotherBox   [Hard]  3.7 GB
https://mega.nz/folder/XXXXXXXX#YYYYYYYYY
```

## Options

```
-f FILE, --file FILE        Links file (default: links.txt)
-d DIR,  --dest DIR         Destination directory (prompted if omitted)
-j N,    --jobs N           Parallel workers (default: 1)
--timeout SECS              Per-file timeout in seconds (default: 7200)
--megacmd-dir DIR           Custom MEGA-CMD path (Windows only)
--select SPEC               e.g. "1", "1,3,5", "2-7", "all"
--level {easy,medium,hard}  Filter by difficulty
--list                      Print link table and exit

HackMyVM integration:
--hmv MACHINE               Machine name to resolve and download
--hmv-user USER             HackMyVM username (or HMV_USER env var)
--hmv-pass PASS             HackMyVM password (or HMV_PASS env var)
```

## Integration with vuln_hmv

```bash
# Step 1: install vuln_hmv
git clone https://github.com/vulnquest58/vuln_hmv.git
cd vuln_hmv && ./setup.sh

# Step 2: export unsolved machine links
vuln_hmv --save-links links.txt -d ~/VMs

# Step 3: batch download
python mega_download.py -f links.txt -d ~/VMs
```
